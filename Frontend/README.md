## ğŸ‘€ Motivation [ğŸ”](#-table-of-contents)

The frontend of Testimonial CMS is built with Next.js, React, and TypeScript, delivering a modern, fast, and scalable interface for managing and displaying testimonials. It leverages Tailwind CSS and Shadcn/ui for a clean and cohesive design, while Zustand handles global state management in a lightweight and predictable way. Zod is used for data validation, and JSON Web Tokens (JWT) manage authentication. Additionally, PostCSS, ESLint, and Prettier ensure clean, accessible, and maintainable code, providing a smooth experience for both administrators and end users.
