## ğŸ“‹ About this project [ğŸ”](#readme-for--restify-app-)

Testimonial CMS is a platform designed for institutions and companies aiming to showcase the real impact of their programs or products through authentic testimonials in text, video, or image format. It provides tools to create, organize, and publish stories with curation, moderation, and engagement analytics, allowing classification by categories, smart search, tagging, and easy sharing via embeds or public API. Built with modern technologies like Next.js or NestJS, PostgreSQL, YouTube API, and Cloudinary, it delivers a robust and flexible system with admin, editor, and viewer roles.
